# Contador de Números Pares e Ímpares

Programa em C que:
- Lê a quantidade de números a serem analisados
- Recebe números inteiros
- Ignora valores menores ou iguais a zero
- Conta quantos números são pares e quantos são ímpares
- Exibe o total de pares e ímpares ao final

## Conceitos utilizados
- Laços de repetição (`for`)
- Estruturas condicionais (`if / else if / else`)
- Operador módulo (`%`)
- Validação de entrada
- Tipos numéricos (`int`)

## Como compilar e executar
```bash
gcc contador_pares_impares.c -o contador
./contador
