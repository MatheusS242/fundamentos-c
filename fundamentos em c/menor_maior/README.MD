# Maior e Menor Valor em Vetor

Programa em C que:
- Lê a quantidade de elementos do vetor
- Recebe números inteiros
- Armazena os valores em um vetor
- Determina o maior e o menor valor informado
- Exibe o menor e o maior número ao final

## Conceitos utilizados

- Vetores
- Laço de repetição (`for`)
- Laço de repetição (`while`)
- Estruturas condicionais (`if`)
- Validação de entrada
- Tipos numéricos (`int`)

## Como compilar e executar

```bash
gcc maior_menor.c -o maior_menor
./maior_menor
